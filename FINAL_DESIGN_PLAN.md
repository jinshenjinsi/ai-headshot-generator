# AI专业头像生成器 - 完整设计方案

## 核心功能定位

**三大功能:**
1. 证照生成(核心功能,刚需,高频)
2. 美照生成(增强功能,创意)
3. 照片修复(扩展功能,超分)

---

## 证照生成流程详细设计

### 第一步: 上传照片
- 用户上传一张正面人物照
- 显示照片预览

### 第二步: 选择证照类型

**分为两类:**

#### A. 有特定要求的证照(需要选择国家)

**护照照:**
- 中国护照 → 35×45mm, 白色背景 ✓ 自动配置
- 美国护照 → 2×2英寸(50.8×50.8mm), 白色背景 ✓ 自动配置
- 日本护照 → 45×45mm, 白色背景 ✓ 自动配置
- 英国护照 → 35×45mm, 白色背景 ✓ 自动配置
- 加拿大护照 → 35×45mm, 白色背景 ✓ 自动配置
- 澳大利亚护照 → 35×45mm, 白色背景 ✓ 自动配置
- 台湾护照 → 35×45mm, 白色背景 ✓ 自动配置
- [更多国家...]

**签证照:**
- 美国签证 → 50×50mm, 白色背景 ✓ 自动配置
- 日本签证 → 45×45mm, 白色背景 ✓ 自动配置
- 欧盟签证 → 48×33mm, 白色背景 ✓ 自动配置
- 英国签证 → 35×45mm, 白色背景 ✓ 自动配置
- 加拿大签证 → 35×45mm, 白色背景 ✓ 自动配置
- 澳大利亚签证 → 35×45mm, 白色背景 ✓ 自动配置
- 印度签证 → 50.8×50.8mm, 白色背景 ✓ 自动配置
- [更多国家...]

#### B. 灵活配置的证照(无需选择国家)

**工作证:**
- 默认尺寸: 1寸(25×35mm)
- 用户可在结果页修改尺寸
- 背景色: 用户自选(白色/蓝色/红色)

**简历照:**
- 默认尺寸: 1寸(25×35mm)
- 用户可在结果页修改尺寸
- 背景色: 用户自选(白色/蓝色)

**学生证:**
- 默认尺寸: 1寸(25×35mm)
- 用户可在结果页修改尺寸
- 背景色: 用户自选(白色/蓝色)

**社交头像:**
- 默认尺寸: 400×400px
- 用户可在结果页修改尺寸
- 背景色: 用户自选(白色/蓝色/红色/透明)

### 第三步: 配置参数

**对于有特定要求的证照(A类):**
```
用户选择类型(护照/签证) 
  → 显示国家列表(热门国家优先排序)
  → 用户选择国家
  → 自动配置尺寸和背景
  → 跳过配置步骤
  → 直接进入编辑
```

**对于灵活配置的证照(B类):**
```
用户选择类型(工作证/简历照等)
  → 显示背景颜色选择
  → 用户选择背景颜色
  → 确认默认尺寸(可在结果页修改)
  → 进入编辑
```

### 第四步: 编辑调整

**编辑功能:**
- 亮度调整(0-200%)
- 对比度调整(0-200%)
- 饱和度调整(0-200%)
- 锐度调整(0-200%)
- 重置按钮

**预览:**
- 实时显示编辑效果

### 第五步: 生成结果

**结果页显示:**
- 最终证照预览
- 证照信息(类型、尺寸、背景)

**下载选项:**
- 当前尺寸下载
- 其他常用尺寸下载(可选)
- 全部尺寸下载(可选)

**尺寸修改功能:**
- 用户可以修改尺寸(宽度、高度)
- 支持自定义尺寸输入
- 修改后重新预览
- 修改后重新下载

**其他操作:**
- 重新编辑(回到编辑页)
- 重新选择(回到选择页)
- 返回首页

## 美照生成流程

### 流程:
1. 上传照片
2. 选择风格(10种风格)
3. 编辑调整(亮度、对比度、饱和度、锐度)
4. 结果展示和下载

### 10种风格:
1. 专业(Professional)
2. 油画(Oil Painting)
3. 水彩(Watercolor)
4. 素描(Sketch)
5. 卡通(Cartoon)
6. 动漫(Anime)
7. 极简(Minimalist)
8. 复古(Vintage)
9. 霓虹(Neon)
10. 艺术(Artistic)

---

## 照片修复流程

### 流程:
1. 上传照片
2. 选择超分倍数(2倍或4倍)
3. 结果展示(修复前后对比)
4. 下载

## 界面设计规范

### 配色方案:
- **主色:** 深蓝色(#1A365D) - 专业、信任、稳重
- **辅色:** 金色(#D4AF37) - 高端、品质感
- **背景:** 白色/浅灰(#F8F9FA) - 简洁、清爽
- **文字:** 深灰(#2D3748) - 易读、专业

### 设计原则:
- 简洁清晰 - 避免复杂的设计
- 高端大气 - 体现品质感
- 用户友好 - 直观的交互流程
- 响应式设计 - 支持各种屏幕尺寸
