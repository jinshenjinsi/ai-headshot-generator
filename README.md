# AI专业头像生成器

一款使用AI技术生成专业商务头像的移动应用,基于React Native + Expo开发,支持iOS和Android平台。

## 功能特点

- **快速上传**: 只需5-10张个人照片即可开始
- **多种风格**: 提供办公室、户外、工作室等6种专业场景
- **AI生成**: 使用内置AI图像生成技术,30-60秒快速生成
- **高清下载**: 支持将生成的头像保存到相册
- **简洁界面**: 遵循iOS人机界面指南,操作流畅直观

## 使用指南

### 1. 上传照片

- 点击首页"开始创建头像"按钮
- 从相册选择5-15张个人照片
- 建议照片光线充足、面部清晰、包含多个角度
- 避免戴帽子、墨镜或有其他人出现

### 2. 选择风格

从以下6种专业风格中选择:

**办公室系列**
- 会议室: 专业会议室背景,正式商务风格
- 办公大堂: 现代办公楼大堂,简约专业

**户外系列**
- 城市街景: 都市建筑背景,时尚现代
- 公园绿地: 自然绿色背景,轻松亲和

**工作室系列**
- 纯白背景: 经典白色背景,简洁专业
- 灰色背景: 中性灰色背景,沉稳大气

### 3. 生成头像

- 点击"生成预览"按钮
- AI将在30-60秒内生成您的专业头像
- 可以查看进度和实时提示

### 4. 下载使用

- 查看生成的头像效果
- 点击"下载高清版"保存到相册
- 支持重新生成或返回首页创建新头像

## 技术架构

### 前端技术栈

- **React Native 0.81**: 跨平台移动应用框架
- **Expo SDK 54**: 开发工具和原生功能集成
- **TypeScript**: 类型安全的开发体验
- **NativeWind**: Tailwind CSS样式方案
- **tRPC**: 类型安全的API调用

### 后端能力

- **内置AI图像生成**: 无需外部API密钥
- **Express服务器**: 处理API请求
- **图像处理**: 支持照片上传和生成

### 核心依赖

- expo-image-picker: 照片选择
- expo-media-library: 相册保存
- expo-haptics: 触觉反馈
- @tanstack/react-query: 数据管理

## 项目结构

```
app/
  (tabs)/
    index.tsx          ← 首页
    gallery.tsx        ← 历史记录
  upload.tsx           ← 照片上传
  style-selection.tsx  ← 风格选择
  generating.tsx       ← AI生成中
  result.tsx           ← 结果展示
lib/
  app-context.tsx      ← 应用状态管理
  trpc.ts              ← API客户端
server/
  routers.ts           ← API路由定义
components/
  screen-container.tsx ← 页面容器
  ui/                  ← UI组件
```

## MVP功能范围

本版本为最小可行产品(MVP),包含核心功能:

✅ 照片上传和管理
✅ 6种预设风格选择
✅ AI图像生成
✅ 结果展示和下载
✅ 本地数据存储

暂不包含:
❌ 用户账号系统
❌ 云端数据同步
❌ 支付系统
❌ 高级编辑功能
