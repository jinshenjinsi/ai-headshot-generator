# 元一图灵 - AI专业头像生成器

一张照片，多样风格。专业证件照在家搞定。

**元一图灵**是一款基于AI技术的专业头像生成应用，提供证件照生成、艺术风格转换和照片超分辨率功能。使用React Native + Expo开发，支持iOS和Android平台。

## 核心功能

### 🎫 证照生成
- **快速生成**：上传一张正面照，AI智能裁剪和优化
- **多种用途**：支持护照、签证、工作证、驾照等多种证件类型
- **标准规格**：自动调整光线、肤色和背景，符合各国标准
- **灵活尺寸**：支持多种证件照尺寸和背景色

### 🎨 美照生成
- **10种专业风格**：油画、水彩、素描、卡通、动漫、极简、黑白、复古等
- **一键转换**：将普通照片变成艺术作品
- **风格预览**：实时预览不同风格效果
- **适用场景**：社交媒体头像、专业用途、创意分享

### 🔍 照片修复
- **超分辨率**：支持2倍和4倍超分辨率，让模糊照片清晰一新
- **智能补全**：AI智能补全细节，提升清晰度
- **老照片修复**：去除噪点、提升色彩，让珍贵回忆焕然一新
- **无损放大**：照片放大不失真

### ✨ 图片调整
- **亮度调整**：精细控制图片亮度
- **对比度调整**：增强或降低对比度
- **饱和度调整**：调整色彩饱和度
- **锐度调整**：增强或降低图片锐度
- **实时预览**：调整时实时显示效果

## 使用指南

### 快速开始

1. **打开应用**
   - 首页展示三大核心功能
   - 选择需要的功能模块

2. **上传照片**
   - 点击对应功能按钮
   - 从相册选择照片或拍照
   - 建议照片光线充足、面部清晰

3. **生成处理**
   - 等待AI处理（通常30-120秒）
   - 实时查看进度
   - 支持随时取消

4. **调整优化**
   - 在结果页面调整亮度、对比度等参数
   - 实时预览调整效果
   - 满意后保存下载

5. **分享下载**
   - 点击分享按钮分享到社交媒体
   - 保存高清版到相册
   - 支持重新生成

## 技术架构

### 前端技术栈

- **React Native 0.81**: 跨平台移动应用框架
- **Expo SDK 54**: 开发工具和原生功能集成
- **TypeScript 5.9**: 类型安全的开发体验
- **NativeWind 4**: Tailwind CSS样式方案
- **tRPC**: 类型安全的API调用
- **React Query**: 数据管理和缓存
- **Reanimated 4**: 流畅的动画效果

### 后端技术栈

- **Node.js + Express**: API服务器
- **tRPC**: 类型安全的RPC框架
- **Alibaba Cloud Bailian**: AI图像生成和处理
- **PostgreSQL**: 数据存储
- **Drizzle ORM**: 数据库操作

### 核心依赖

- **expo-image-picker**: 照片选择
- **expo-media-library**: 相册保存
- **expo-haptics**: 触觉反馈
- **expo-file-system**: 文件系统操作
- **react-native-gesture-handler**: 手势识别
- **@tanstack/react-query**: 数据管理

```
app/
  (tabs)/
    index.tsx              ← 首页

  repair-upload.tsx       ← 修复上传
  repair-generating.tsx   ← 修复生成中
  repair-result.tsx       ← 修复结果
components/
  screen-container.tsx    ← 页面容器
  feedback-modal.tsx      ← 反馈表单
  ui/                     ← UI组件库
lib/
  trpc.ts                 ← API客户端
  theme-provider.tsx      ← 主题管理
hooks/
  use-app-update.ts       ← 应用更新检查
  use-colors.ts           ← 主题颜色
server/
  routers.ts              ← API路由定义
  bailian-service.ts      ← AI服务集成
  _core/
    index.ts              ← 服务器入口
    systemRouter.ts       ← 系统API路由
```

## 开发说明

### 环境要求

- Node.js 18+
- pnpm 9.12.0+
- Expo CLI 最新版本
- Android SDK 或 Xcode（真机测试）

### 安装依赖

```bash
pnpm install
```

### 启动开发服务器

```bash
# 启动前后端开发服务器
pnpm dev

# 或分别启动
pnpm dev:server    # 后端服务器
pnpm dev:metro     # 前端开发服务器
```

**使用Expo Go（推荐快速测试）**


## 常见问题

### Q: 生成失败怎么办？
A: 请检查网络连接，确保后端服务器正常运行。如果问题持续，请通过"意见反馈"功能报告。

### Q: 支持哪些照片格式？
A: 支持JPG、PNG、HEIC等常见格式。建议使用JPG格式以获得最佳效果。

### Q: 生成的照片可以商用吗？
A: 可以。生成的照片版权归用户所有。

### Q: 如何更新应用？
A: 应用会自动检查更新。如有新版本，首页会显示更新提示。
