# AI专业头像生成器 - 移动应用设计文档

## 应用概述

一款使用AI技术生成专业商务头像的移动应用,用户只需上传5-10张个人照片,选择喜欢的风格和场景,即可快速生成高质量的专业头像。

## 设计原则

- **移动优先**: 专为移动端设计,竖屏9:16比例,适配单手操作
- **简洁直观**: 遵循iOS人机界面指南,界面清晰,操作流畅
- **快速反馈**: 提供即时预览,用户无需长时间等待
- **视觉吸引**: 使用渐变色和现代化UI设计,提升用户体验

## 核心屏幕设计

### 1. Home Screen (首页)
**主要内容**:
- 应用Logo和欢迎标题
- 简短的功能介绍文案
- 大型CTA按钮:"开始创建头像"
- 底部展示示例头像轮播

**用户流程**: 
用户打开应用 → 查看介绍 → 点击"开始创建" → 进入上传页面

### 2. Upload Screen (照片上传页面)
**主要内容**:
- 顶部进度指示器(步骤1/3)
- 标题:"上传您的照片"
- 说明文字:"至少上传5张照片,建议10张以获得最佳效果"
- 照片网格展示区(3列布局)
- 每个照片卡片显示缩略图和删除按钮
- 底部"添加照片"按钮(相机图标)
- "下一步"按钮(至少5张照片后激活)

**功能要求**:
- 支持从相册选择多张照片
- 支持拍照添加
- 照片质量检测和提示
- 最少5张,最多15张限制

**用户流程**:
点击"添加照片" → 选择相册/拍照 → 选择多张照片 → 预览照片网格 → 点击"下一步"

### 3. Style Selection Screen (风格选择页面)
**主要内容**:
- 顶部进度指示器(步骤2/3)
- 标题:"选择您的风格"
- 场景分类标签(办公室、户外、工作室、创意)
- 风格卡片网格(2列布局)
- 每个卡片显示:
  - 示例图片
  - 风格名称
  - 简短描述
  - 选中状态指示器
- 底部"生成预览"按钮

**风格选项**:
- **办公室系列**: 会议室、大堂、办公桌
- **户外系列**: 城市街道、公园、建筑外景
- **工作室系列**: 纯色背景、专业灯光
- **创意系列**: 咖啡厅、图书馆、艺术空间

**用户流程**:
浏览风格卡片 → 点击选择风格 → 点击"生成预览"

### 4. Generating Screen (生成中页面)
**主要内容**:
- 居中的加载动画(渐变色旋转圆环)
- 标题:"AI正在生成您的专业头像"
- 进度文字:"预计需要30秒..."
- 动态提示文字(每5秒切换):
  - "正在分析您的面部特征..."
  - "正在应用专业光线效果..."
  - "正在优化背景场景..."
  - "即将完成..."

**功能要求**:
- 真实的AI生成进度反馈
- 不可取消(避免浪费资源)
- 生成完成后自动跳转

### 5. Result Screen (结果展示页面)
**主要内容**:
- 大图展示区(占屏幕60%)
- 生成的头像全屏显示
- 左右滑动切换不同风格(如果生成多个)
- 底部操作按钮组:
  - "下载高清版"(主要按钮)
  - "重新生成"(次要按钮)
  - "分享"图标按钮
- 图片信息:风格名称、生成时间

**用户流程**:
查看生成结果 → 左右滑动对比 → 选择满意的 → 下载/分享

### 6. Gallery Screen (历史记录页面)
**主要内容**:
- 标题:"我的头像"
- 照片网格展示(3列布局)
- 每个卡片显示缩略图和生成日期
- 点击进入详情查看
- 支持删除和分享

**用户流程**:
从底部Tab进入 → 浏览历史记录 → 点击查看详情 → 下载/分享/删除

## 底部Tab导航

| Tab | 图标 | 标题 | 功能 |
|-----|------|------|------|
| Home | house.fill | 首页 | 开始创建新头像 |
| Gallery | photo.fill | 我的 | 查看历史记录 |

## 颜色方案

### 主色调
- **Primary**: `#6366F1` (靛蓝色 - 专业、可信赖)
- **Primary Dark**: `#4F46E5`

### 背景色
- **Light Background**: `#FFFFFF`
- **Dark Background**: `#0F172A`
- **Surface Light**: `#F8FAFC`
- **Surface Dark**: `#1E293B`

### 文字色
- **Foreground Light**: `#0F172A`
- **Foreground Dark**: `#F1F5F9`
- **Muted Light**: `#64748B`
- **Muted Dark**: `#94A3B8`

### 功能色
- **Success**: `#10B981` (绿色)
- **Warning**: `#F59E0B` (橙色)
- **Error**: `#EF4444` (红色)

## 字体规范

- **标题**: 24-32px, 粗体
- **副标题**: 18-20px, 半粗体
- **正文**: 14-16px, 常规
- **说明文字**: 12-14px, 常规

## 交互设计

### 按钮反馈
- **主要按钮**: 按下时缩放至0.97 + 轻微触觉反馈
- **卡片**: 按下时透明度降至0.7
- **图标按钮**: 按下时透明度降至0.6

### 动画
- **页面切换**: 250ms淡入淡出
- **卡片选中**: 边框高亮 + 轻微缩放
- **加载动画**: 渐变色旋转圆环

## 技术实现要点

### MVP功能范围
1. ✅ 照片上传(相册选择)
2. ✅ 风格选择(4-6个预设风格)
3. ✅ AI生成(Stable Diffusion API)
4. ✅ 结果展示和下载
5. ✅ 本地历史记录(AsyncStorage)

### 暂不包含
- ❌ 用户账号系统
- ❌ 云端同步
- ❌ 支付系统
- ❌ 高级编辑功能

## 用户体验优化

1. **快速上手**: 首次使用无需注册,直接开始创建
2. **即时反馈**: 每个操作都有视觉和触觉反馈
3. **进度可见**: 清晰的步骤指示器
4. **容错设计**: 照片数量不足时提示,质量问题时警告
5. **本地优先**: 所有数据本地存储,无需网络即可查看历史记录
